Customized Prototype Writeup

  • Ultimately, I plan to turn in a design that centers around sharing locations: an embedded Google Maps that will take you there, an input for a location in your message that automatically performs that search when clicked, a display of the location on the Map in chat, and a clickable list of the recent locations mentioned in the chat.
  • At the core, it is similar to my Figma mockup (generic embedded Maps search and automatic specific location search), but it includes a separate input for the location instead of making every message queryable. This makes it more intuitive, safer, and easier to discover for the user. I also decided to include a list of recent locations mentioned in the chat for users to easily scan through and click on.
  • This week, I have completed the Google Maps icon generic search, the location input that automatically takes you there on Google Maps, and the clickable list of recent locations. I manually added the embedded maps for a few locations to be displayed in the chat for a proof of concept (try typing in cupertino, jonquils, stata, or mit in the location input). Google Maps API was not working for me (or I would need to pay) and the iFrame had very specific inputs to make it work. The TAs suggested I do this method for now, but if there is another way I could display the maps of the locations, I'd be delighted to know. Otherwise, I have completed the design I set out to do. Please let me know if I need to add more.
